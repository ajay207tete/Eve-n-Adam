# Eve n Adam - Simple TON dApp

A basic demo dApp for TON blockchain that:
- Connects to a TON wallet
- Sends 1 TON to the owner address

### Features
- Simple UI
- Wallet connection
- Payment transaction

### Tech Stack
- HTML, CSS, JS
- TON Connect UI

### How to Run
1. Open `index.html` in browser.
2. Connect wallet.
3. Click **Pay 1 TON** to send transaction.
